# Kütüphanem

Hikaye okuma ve yazma platformu.

## Kurulum

```bash
npm install
npm start
```

## Deploy Etme

### Vercel (Önerilen)
1. [vercel.com](https://vercel.com) üzerinde hesap aç
2. "Add New Project" → "Upload" seç
3. Bu ZIP dosyasını yükle
4. "Deploy" butonuna bas — bitti!

### Netlify
1. [netlify.com](https://netlify.com) üzerinde hesap aç
2. "Add new site" → "Deploy manually"
3. Bu ZIP dosyasını sürükle-bırak
4. Otomatik deploy olur!

## Derleme

```bash
npm run build
```
